SnagList App Specification

Application Description

Record defects in a job/project. Send defects report to contractors in form of cases. Get feedback from contractors on each case. Close repair case or request resubmission.

Front End Sequence

  • Login / Register Page
  • Load or create a job/project
    • Create / Change Project Name
    • Add/Remove Project Members (Contractors / Inspectors)
  • Create New Snag
    • Add / Remove Snag Photos
    • Add / Remove Snag Documents
    • Add / Create / Edit Location
    • Add / Create / Edit Snag Description
    • Add / Create Contractors
    • Add / Create Inspectors
    • Expected Rectification Date
  • Generate/Send Snag Report to Contractors/Inspectors
    • List of All Snags With their States
      • In Progress
      • Repaired (Waiting for Inspectors)
      • Completed
  • View list of Snags
    • By State
    • By Contractor
    • By Inspector
  • List My Snags
  • Contractor Snag Repair
    • Repaired Snag Photo
    • Repair Description
